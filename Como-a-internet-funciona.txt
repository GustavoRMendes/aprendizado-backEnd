RESUMO: Como a internet funciona?

O que é a Internet? A internet é uma rede global de computadores interconectados que permite a troca de informações e utilização de diversos serviços. É feita de cabos que passam por debaixo da terra(mar), e que conectam casas, países e continentes.
-------------------------------------------------------------------------
TCP/IP (Transmission Central Protocol/ Internet Protocol )
Protocolo é uma forma padronizada de realizar determinadas ações e formatar dados para que dois ou mais dispositivos sejam capaz de ser comunicar entre si.
Todo dispositivo conectado a rede recebe um IP para que possam ser identificados , como se fossem um CEP.
Cada Página na internet possui um IP, porém seria difícil memorizar o IP de cada página, por isso eles tem endereços memoráveis, ou seja, nomes.

ISP -> PROVEDOR DE Internet
PROTOCOLO -> CONJUNTO DE REGRAS, IDENTIFICA CADA DISPOSITIVO E DEPOIS DIVIDE A INFORMAÇÃO EM PACOTES.
DNS -> DOMAIN NAME SYSTEM ("AGENDA ELETRÔNICA")

Um pacote é um pequeno segmento de uma mensagem maior de cada pacote, contém dados quanto informações sobre os mesmos.
O conteúdo do pacote é conhecido como cabeçalho.
Quando os dados são enviados pela internet, eles primeiro são divididos em pacotes, que depois são traduzidos em bits.
Os pacotes são encaminhados por vários dispositivos de rede, tais como roteadores e switches.
Chega em pedaços e o dispositivo receptor monta novamente na ordem correta.
É um processo extremamente rápido.

  Comutação de pacotes 
    Os roteadores e switches conseguem processar pacotes de forma independente da origem ou do destino. Isso é concebido para que nenhuma conexão única domine a rede.

HTTP E HTTPS
Para acessar uma página, utilizamos o protocolo http (HyperText Transfer Protocol) que é um conjunto de regras que permitem seu computador trocar informações com um servidor que abriga uma página.
HTTP -> Define como são requisitadas as páginas da WEB, baseada em texto, o que pode ser alterado ou interceptado. Feito para comunicação e não a privacidade.

HTTPS -> Insere uma camada de proteção na transmissão de dados entre seu computador e servidor. É criptografada a comunicação.
Também faz a autenticação das páginas.
